# Breadcrumb

```vue
<Breadcrumb>
    <BreadcrumbItem>
        Home
    </BreadcrumbItem>

    <BreadcrumbItem active>
        Products
    </BreadcrumbItem>
</Breadcrumb>
```

## Slots

| Name    | Description                            | Bindings |
| ------- | -------------------------------------- | -------- |
| default | Displays content inside the breadcrumb |          |
