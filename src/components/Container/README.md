# Container

```vue
<Container>
    <Row>
        <Col>
            <h1>Hello World</h1>
        </Col>
    </Row>
</Container>
```

## Props

| Prop name | Description                      | Type    | Values | Default |
| --------- | -------------------------------- | ------- | ------ | ------- |
| fluid     | Uses the `container-fluid` class | boolean | -      | false   |

## Slots

| Name    | Description                           | Bindings |
| ------- | ------------------------------------- | -------- |
| default | Displays content inside the container |          |
