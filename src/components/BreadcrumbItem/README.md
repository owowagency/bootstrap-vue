# BreadcrumbItem

```vue
<BreadcrumbItem active>
    Products
</BreadcrumbItem>
```

## Imported Props

| Prop name | Description             | Type    | Values | Default |
| --------- | ----------------------- | ------- | ------ | ------- |
| active    | Adds the `active` class | boolean | -      | false   |

## Slots

| Name    | Description                                 | Bindings |
| ------- | ------------------------------------------- | -------- |
| default | Displays content inside the breadcrumb item |          |
